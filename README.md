# Calculadora de IMC (calc-imc)

Aplicação Java com interface gráfica Swing simples para calcular o Índice de Massa Corporal (IMC).

## Requisitos
Tenha o pacote `default-jdk` instalado

## Como usar
1. Clone o repositório para um diretório local:
   ```bash
   git clone https://github.com/gabrielff5/calc-imc.git
   ```
1. Compile o projeto:
   ```bash
   javac CalculadoraIMC.java
   ```
